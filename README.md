### Authentication
- Email/Password registration & login
- Google Sign-In (Firebase)
- Facebook Login
- Email OTP verification
- JWT token-based authentication

### Backend (Node.js + Express)
- RESTful API
- MySQL database
- Email sending (nodemailer)
- OAuth integration
- Password hashing (bcrypt)
